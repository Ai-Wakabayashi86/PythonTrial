# ToDoリスト

## 作成目的
タスク管理のためのシステムを作る

## 必要な機能
- GUI(画面上で操作する機能)
- データベース(各タスクを記録しておく機能)
- アラーム(現在時刻から計算してからアラームを設定する機能)

## 構成
- ホーム
  - (例)仕事
    - タスクの新規作成
      - タスク名・アラーム時間の設定
    - タスクの変更
      - タスク名・アラーム時間の変更
    - タスクの完了
    - タスクの削除
  - (例)趣味
    - タスクの新規作成
      - タスク名・アラーム時間の設定
    - タスクの変更
      - タスク名・アラーム時間の変更
    - タスクの完了
    - タスクの削除
  - (例)子供
    - タスクの新規作成
      - タスク名・アラーム時間の設定
    - タスクの変更
      - タスク名・アラーム時間の変更
    - タスクの完了
    - タスクの削除

## 使用言語
- GUI：JavaScript/Python
- DB：SQL
